WebMind

September 2017 - December 2017

Description

WebMind is a desktop application that allows a user to "play" Conway's Game of Life. The application, first, presents the user the with a grid where he or she can declare the initial state of the simulation. The user is then able to run the simulation and watch the board evolve. The application's GUI was written using Java Swing and the engine in C++. A script was also developed, in C++, that allows a user to initialize a board, but he or she cannot run the simualtion from the script.

My Role

For WebMind, I worked with other classmates to create the GUI in Java Swing. I also worked with the Honors College Group to create the script for the application. The script group was smaller so I was able able to work more intimately with that part of the project. We created a recursive decent parser in C++ to parse through script commands and create JSON strings. The JSON strings contain commands for the engine in a schema that the engine group had predetermined.

Tools and Languages

Java/Java Swing, C++, Git and Git Lab, Linux
